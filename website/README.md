# slay website

The marketing site for [slay](https://slay.fyniti.co.uk) — built with [Astro](https://astro.build).

## Development

```bash
# install dependencies
npm install

# start dev server
npm run dev
```

The dev server runs on **port 4321** by default:

```
   SLAY  dev server ready

  ├─ Local     http://localhost:4321/
  ├─ Network   http://192.168.x.x:4321/
  ├─ Port      4321
  ├─ Env       development
  ╰─ Started   02:30:15 PM
```

## Scripts

| Command           | Description                |
| ----------------- | -------------------------- |
| `npm run dev`     | Start dev server `:4321`   |
| `npm run build`   | Build for production       |
| `npm run preview` | Preview production build   |
| `npm run test`    | Run tests with Vitest      |
