# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Email:** hammadkhanxcm@gmail.com

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

We will acknowledge your report within 48 hours and aim to provide a fix within
7 days for critical issues.

## Scope

Security concerns for this project include:

- Command injection via port arguments
- Privilege escalation through process killing
- Unexpected behavior with elevated permissions

Please do **not** open a public issue for security vulnerabilities.
